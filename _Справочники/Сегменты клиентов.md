# Сегменты клиентов (теоретические)

> **← Хаб**

10 сегментов в трёх крупных группах. Используются как **интерфейсы Слоя 3**: для каждого определены ожидаемые поведенческие метрики (Δ-метрики).

> [!warning] **Текущий статус — теоретический**
> Сегменты и метрики используют стандартные дефолты из CCAR / EBA stress test methodology + open-source банковской аналитики. Реальные сегменты Т-Банка могут отличаться по границам и метрикам — будут заменены при подключении внутренних данных в фазе 3.

## ФЛ — Физические лица (4 сегмента)

### ФЛ.1 Массовый

- **Кто:** клиент со среднемесячным остатком < 100 тыс. ₽; средний доход < 80 тыс. ₽/мес
- **Доля портфеля:** ~70–80% количества; ~25–30% AUM
- **Ключевые продукты:** дебетовая карта, потребкредит, рассрочка/BNPL, депозит до 1 млн ₽
- **Чувствительность к шокам:**
  - КС ↑↑ → отказ от потребкредитов (high), переход в депозиты, рост дефолтов по картам
  - Война / эскалация → рост спроса на кэш, валюту, отток в наличные
  - Климат → локальные эффекты по регионам
  - Кризис → массовое снижение depositов, рост просрочки

**Δ-метрики (выходные сигналы Слоя 3):**
- ΔPD (12М) по потребкредитам и кредиткам
- Δ-отклонение в розничной активности (LFL транзакций)
- Δ-чистый отток / приток депозитов
- Δ-доля просрочки 30+ дней
- Δ-LTV для ипотечных активов сегмента

### ФЛ.2 Средний класс

- **Кто:** среднемесячный остаток 100 тыс. – 1 млн ₽; доход 80–250 тыс. ₽/мес
- **Доля портфеля:** ~15–20% количества; ~35–40% AUM
- **Ключевые продукты:** ипотека, автокредит, инвестсчёт, накопительные счета, ИИС
- **Чувствительность к шокам:**
  - КС ↑↑ → переоценка ипотеки (риск рефинансирования), отток из инвестсчетов в депозиты
  - Война → волатильность инвестпортфелей, отток в защитные активы
  - Кризис → задержка платежей по ипотеке, потеря работы → дефолты с лагом
  - Климат → региональные ипотечные риски (затопления → переоценка залога)

**Δ-метрики:**
- ΔPD по ипотеке / автокредитам
- Δ-объём новых заявок на ипотеку
- Δ-AUM в инвестпродуктах (отток / приток)
- Δ-доля рефинансирования

### ФЛ.3 Премиум

- **Кто:** среднемесячный остаток 1–10 млн ₽; доход 250–1000 тыс. ₽/мес
- **Доля портфеля:** ~3–5% количества; ~20–25% AUM
- **Ключевые продукты:** премиум-карта, инвестсоветники, валютные счета, ипотека на квартиры > 20 млн ₽, structured products
- **Чувствительность к шокам:**
  - КС ↑↑ → перебалансировка в облигации/сбережения (умеренная)
  - Война → активное движение в валюту, золото, недвижимость
  - Кризис → защитные стратегии (cash, gold)
  - Климат → mало прямого эффекта

**Δ-метрики:**
- Δ-AUM (приток в защитные / отток в спекулятивные)
- Δ-доля валютных депозитов
- Δ-активность в инвест-сегменте

### ФЛ.4 Private Banking

- **Кто:** AUM > 10 млн ₽; высокообеспеченные клиенты, бизнес-владельцы
- **Доля портфеля:** ~0,5–1% количества; ~15–20% AUM
- **Ключевые продукты:** wealth management, private placements, advisory, family offices
- **Чувствительность к шокам:**
  - Война / санкции → главный канал; жёсткие санкции на российских HNWI закрывают зарубежные структуры
  - Кризис → защита капитала через структурирование
  - КС → второстепенно (короткие позиции / leverage)
  - Климат → минимально

**Δ-метрики:**
- Δ-AUM глобальный
- Δ-структура (Россия / зарубежье)
- Δ-leverage
- Δ-доля «токсичных» активов (под санкциями)

## SME — Малый и средний бизнес (3 сегмента)

### SME.1 Микро (≤ 100 млн ₽ годовая выручка)

- **Кто:** ИП, малые ООО; персонал ≤ 15 чел.
- **Доля портфеля:** ~80% количества; ~10–15% портфеля по объёму
- **Ключевые продукты:** оборотный кредит, эквайринг, зарплатные проекты, BNPL для покупателей
- **Чувствительность к шокам:**
  - КС ↑↑ → катастрофически (нет финансовой подушки, плавающие ставки кредитов) → массовые дефолты в течение 2–4 кварталов
  - Кризис → банкротства (сегмент с самой высокой PD)
  - Война → региональные эффекты (мобилизация рабочей силы)
  - Климат → прямой удар по бизнесу в зоне катастрофы

**Δ-метрики:**
- Δ-PD (12М) по сегменту (самая высокая чувствительность)
- Δ-выручка по эквайрингу (proxy для деловой активности)
- Δ-новые открытия / закрытия счетов
- Δ-объём кредитного спроса

### SME.2 Малый (100–500 млн ₽ годовая выручка)

- **Кто:** ООО среднего размера; 15–100 чел.
- **Доля портфеля:** ~15% количества; ~25% по объёму
- **Ключевые продукты:** инвесткредиты, кредитные линии, проектное финансирование, ВЭД
- **Чувствительность к шокам:**
  - КС ↑↑ → задержка инвестпроектов, переход в режим выживания
  - Кризис → спад выручки, проблемы с дебиторкой
  - Война → секторальные эффекты (строительство, импорт)
  - Климат → региональные

**Δ-метрики:**
- Δ-PD (12М)
- Δ-объём new originations
- Δ-revolver utilization
- Δ-задержка платежей

### SME.3 Средний (500 млн – 2 млрд ₽ годовая выручка)

- **Кто:** региональные лидеры, сильные ООО / средние ОАО
- **Доля портфеля:** ~5% количества; ~50–60% по объёму
- **Ключевые продукты:** структурированное финансирование, торговое финансирование, валютные операции
- **Чувствительность к шокам:**
  - Все 4 категории — заметно, но обычно с большим лагом (1–3 кв.)
  - Близок к нижней границе M+/L клиентов

**Δ-метрики:**
- Δ-PD (12М)
- Δ-EBITDA сегмента (агрегат)
- Δ-coverage ratio
- Δ-NPL transition

## M+/L — Корпоративные клиенты (3 сегмента)

### M+.1 Mid-corporate (выручка 2–50 млрд ₽)

- **Кто:** региональные лидеры отраслей, не входящие в топ-100
- **Доля портфеля:** ~20% количества; ~30% по объёму M+L
- **Ключевые продукты:** многоэлементное финансирование, M&A advisory, ECM/DCM
- **Чувствительность к шокам:**
  - КС → через рефинансирование плавающего долга
  - Кризис → отраслевая зависимость
  - Война → санкционный риск
  - Климат → отраслевая специфика

**Δ-метрики:**
- Δ-EBITDA / ICR / ЧД/EBITDA
- Δ-Probability of rating downgrade
- Δ-новые заявки на финансирование
- Δ-CDS-spread (если есть рыночное finansirование)

### L.1 Large-corporate (выручка 50+ млрд ₽)

- **Кто:** топ-50 заёмщиков РФ; публичные эмитенты с рейтингами; госкорпорации
- **Доля портфеля:** ~70% объёма M+L; концентрация ~5% клиентов
- **Ключевые продукты:** проектное финансирование, синдицированные кредиты, евробонды/юанебонды, ECM/DCM
- **Чувствительность к шокам:**
  - Война / санкции → прямой; SDN-санкции на конкретного эмитента
  - Кризис → затяжной (через дочки и каскад)
  - КС → через стоимость нового долга
  - Климат → отраслевая

**Δ-метрики:**
- Δ-кредитный рейтинг (АКРА / Эксперт РА / Moody's / S&P / Fitch — где есть)
- Δ-CDS / cash bond spread
- Δ-капитализация (для публичных)
- Δ-операционные показатели (выручка, EBITDA, FCF)

### Public.1 Публичные эмитенты с рейтингами (срез поверх M+/L)

- **Кто:** компании, чьи бумаги торгуются + есть открытый кредитный рейтинг
- **Особенность:** для них **есть рыночные сигналы** (CDS, акции, бонды) → высокая частота обновления данных
- **Δ-метрики:**
  - Δ-CDS spread (если торгуется)
  - Δ-bond yield spread к ОФЗ
  - Δ-волатильность акции
  - Δ-implied default probability (KMV-стиль)

## Многомерные срезы — теги поверх 10 базовых сегментов

> **Дополнение v1.1 (2026-04-25):** базовое деление по AUM/выручке — недостаточно. Реальная реакция на шок зависит **от пересечения** базового сегмента с другими измерениями: регион, возраст, профессия, тип продукта, валютная экспозиция. Ниже — 5 ортогональных размерностей, которые комбинируются как теги.

Каждый клиент описывается как: `базовый_сегмент × регион × возраст × профессия × продукт × валюта`. Например: «ФЛ.Средний × Москва × 36-50 × IT × ипотека-плавающая × частично-USD-доход».

### Размерность 1 — Регион (двухуровневая v1.2)

> **Изменение v1.2 (2026-04-25):** двухуровневая иерархия. Верхний уровень — макро-регион (8 тегов); нижний уровень — **микро-региональный тип** (5 тегов). Реальный сегмент описывается как **MACRO × MICRO**, например `SOUTH_CAUCASUS × URBAN_INDUSTRIAL`.
>
> **Триггер для введения:** анализ Туапсе показал бифуркацию `Краснодарский край × URBAN_INDUSTRIAL` (Туапсе город — HH ущерб) vs `Краснодарский край × TOURIST_RESORT` (Сочи/Геленджик/Анапа — +10% бронирований).

#### Уровень 1 — Макро-регион

| Тег | Состав | Особенности | Чувствительность |
|---|---|---|---|
| **MSK_SPB** | Москва, СПб | Высокая ср.зарплата, дорогая ипотека, премиум-проникновение | КС ↑ → ипотека (HH); инфляция ↓ реальные доходы (M) |
| **DONOR** | ХМАО, Тюмень, Сахалин, Татарстан, Ямал | Бюджет региона ≥50% от нефтегаза; региональные надбавки к зарплатам | Brent ↓ → бюджетные индексации замедл. (H); SDN на ВИНК (HH) |
| **INDUSTRIAL** | Свердловская, Челябинская, Кемеровская, Магадан, Липецкая, Вологодская | Зависят от промышленных корпораций (металлурги, машиностроение, добыча) | Кризис чёрной металлургии (HH); санкции SDN на корпорацию региона (H) |
| **SOUTH_CAUCASUS** | Дагестан, Краснодар, Ставрополь, Ростов | Аграрный + туристический + транзитные доходы | Климат-шоки (HH); туристические ограничения (H); переселение из зон СВО (M) |
| **FAR_EAST** | Приморский, Хабаровский, Сахалин, Камчатка | Высокие региональные надбавки + связь с Азией; импорт из КНР дешевле | КНР-санкции на экспорт (H); ослабление рубля → импорт дороже (M) |
| **SIBERIA** | Кр-ярский, Иркутская, Новосиб., Кемерово, Бурятия | Алюминий/уголь/металлы; региональная экономия | Цена меди/угля (H); тарифы Восточного полигона (M) |
| **CENTRAL** | ЦФО без МСК, Поволжье | Смешанная экономика, средняя плотность доходов | Все каналы — средняя чувствительность |
| **RURAL** | Сельские населённые пункты любого региона | Низкие доходы, сильная зависимость от АПК и бюджета | Климат-шоки (HH для АПК); инфляция продовольствия (HH) |

#### Уровень 2 — Микро-региональный тип (новое в v1.2)

| Тег | Описание | Примеры | Особая чувствительность |
|---|---|---|---|
| **TOURIST_RESORT** | Курортные города, основной доход — туризм | Сочи, Геленджик, Анапа, Кисловодск, Ялта, Светлогорск | Туристический поток — главный канал; экологические риски в радиусе 10+ км затрагивают слабее, чем ожидается |
| **URBAN_INDUSTRIAL** | Городские центры с тяжёлой промышленностью или критической инфраструктурой | Туапсе, Новокуйбышевск, Магнитогорск, Норильск, Череповец | Прямые удары по инфраструктуре (атаки, аварии) → жилой сегмент в зоне 5 км ловит HH; SME городского бизнеса — HH простой |
| **MONOTOWN** | Моногорода (≥1 системообразующее предприятие) | Магнитогорск (ММК), Тольятти (АвтоВАЗ), Норильск (Норникель), Новокузнецк (Евраз) | Кризис системообразующего → каскад на весь город. Ипотека и SME привязаны к одному работодателю |
| **AGRICULTURAL_RURAL** | Сельские районы с АПК-фокусом | Кубань (внутренние районы), Ставрополье (зерновая зона), Алтайский край | Климат-шоки (HH); цены продовольствия; госсубсидии АПК; внеоборотные средства зависят от урожая |
| **CAPITAL_DIVERSIFIED** | Региональные столицы со смешанной экономикой | Казань, Краснодар, Екатеринбург, Новосибирск (как города) | Сбалансированный риск-профиль; диверсифицированная база доходов |

**Применение:** для каждой новости с явной географической привязкой указывать **обе размерности**. Если новость затрагивает только **один микротип внутри макро-региона** — это **критично для бифуркации** (как в Туапсе).

**Пример микросегментации Туапсе 20.04.2026:**
- `SOUTH_CAUCASUS × URBAN_INDUSTRIAL` (Туапсе город) → ΔPD ипотека +20..+40%, SME простой 1-3 мес
- `SOUTH_CAUCASUS × TOURIST_RESORT` (Сочи/Геленджик/Анапа) → бронирования +10% г/г = бенефициар
- **Без микротипа: эти два сегмента выглядят одинаково (всё «Краснодарский край») — критичная ошибка**.

#### Известные «горячие» микро-локации (для быстрой идентификации)

| Микро-локация | Макро | Микро | Главный риск |
|---|---|---|---|
| Туапсе | SOUTH_CAUCASUS | URBAN_INDUSTRIAL (НПЗ Роснефти) | Атаки на инфраструктуру; экологические разливы |
| Сочи / Геленджик / Анапа | SOUTH_CAUCASUS | TOURIST_RESORT | Турпоток; курортный сезон; форс-мажор побережья |
| Магнитогорск | INDUSTRIAL | MONOTOWN (ММК) | Кризис чёрной металлургии → весь город |
| Норильск | SIBERIA | MONOTOWN+URBAN_INDUSTRIAL (Норникель) | Цена меди/палладия; экологические штрафы |
| Татарстан (Альметьевск/Нижнекамск) | DONOR | URBAN_INDUSTRIAL (Татнефть/Танеко) | Атаки на НПЗ; санкции на нефть |
| Самарская обл. (Сызрань/Тольятти) | INDUSTRIAL | URBAN_INDUSTRIAL + MONOTOWN | Атаки на НПЗ; кризис АвтоВАЗа |

### Размерность 2 — Возраст

| Тег | Возраст | Особенности | Реакция на шоки |
|---|---|---|---|
| **YOUNG** | 18–25 | Студенты, молодые работающие; BNPL, кредитки, малая ипотека | КС → BNPL/потребкредит (HH); военная мобилизация (HH для м.); без накоплений на стрессовый период |
| **MILLENNIAL** | 26–35 | Молодые семьи, активные ипотечники, инвесторы | КС → новая ипотека (HH); валютные колебания → инвестпортфели (H); рождение детей → потреб.активность |
| **MID_CAREER** | 36–50 | Зрелые карьеры, продолж. ипотеки, активные инвесторы и Премиум | Все каналы — устойчиво умеренная (M). Главный риск — потеря работы при рецессии |
| **PRE_RETIRE** | 51–65 | Пред-пенсионный, накопления → депозиты, продажа активов | КС ↑ → депозиты привлекательны (HH); инфляция ↑ → реальные накопления ↓ (H) |
| **RETIREE** | 65+ | Пенсионеры; депозиты; зависят от индексации пенсий | Инфляция (HH); индексация пенсий (HH); невосприимчивы к КС-каналу |

### Размерность 3 — Профессия / Занятость

| Тег | Состав | Доходовая модель | Чувствительность к шокам |
|---|---|---|---|
| **GOV_BUDGET** | Госслужащие, бюджетники (учителя, врачи, военные служащие, силовики) | Стабильный ₽-доход + индексация | Дефицит бюджета → задержка индексации (H); военные → расширение состава (нейтрально/+) |
| **INDUSTRY_WORKER** | Промышленные рабочие в металлургии/НГ/химии/машиностроении | Зависят от спроса промышленности | Кризис отрасли → сокращения (HH); закрытие предприятия (HH) |
| **SERVICES** | Сфера услуг, общепит, ритейл-персонал | Зависят от потреб.активности | Спад спроса в рецессию (HH); инфляция → реальные доходы (H) |
| **IT_FREELANCE** | IT-сектор, фриланс, удалённая работа | Часто с валютным или гибридным доходом | Санкции на IT-инфраструктуру (H); запрет на платежи (M); ослабление рубля → ↑ доход в ₽ (+) |
| **SELF_EMPLOYED** | Самозанятые, ИП | Самостоятельный нерегулярный доход | КС → дороже оборотный кредит (HH); регулирование самозанятости (H) |
| **MILITARY_CONTRACT** | Военнослужащие по контракту, СВО-выплаты | Выс. ₽-доход + единовременные выплаты | Нейтрально/+: новые контракты ↑; +инвалидизация (отдельная подгруппа) |
| **CORP_EXEC** | Корп.менеджеры M+/L бизнеса | Высокий доход + бонусы (часто валютные) | Санкции против корп. → потеря работы / релокация (HH); премии ↓ при -EBITDA (M) |
| **PENSIONER** | Пенсионеры (см. возраст RETIREE) | Государственная пенсия | Инфляция (HH); индексация (HH) |

### Размерность 4 — Тип основного продукта в банке

| Тег | Особенность | Главный риск-канал |
|---|---|---|
| **MORTGAGE_FIXED** | Ипотека с фиксированной ставкой | Низкая чувствительность к КС текущему; высокая к перепродаже жилья |
| **MORTGAGE_FLOAT** | Ипотека с плавающей ставкой / переменной | КС ↑ → платёж ↑ (HH); риск дефолта в острый стресс |
| **CONSUMER_CREDIT** | Активные заёмщики потребкредитов и кредиток | КС ↑ → дороже refi (HH); инфляция → реальный долг ↓ |
| **DEPOSITOR** | Накопители в депозитах (без долга) | КС ↑ → депозиты выгодны (+); инфляция (-) |
| **INVESTOR** | ИИС, БПИФ, акции, облигации | Волатильность рынка; санкции на брокерскую инфраструктуру |
| **SALARY_ONLY** | Только зарплатный счёт + дебеткарта | Низкая активность; чувствительность через работодателя |
| **BNPL_USER** | Активный пользователь рассрочки | Рост стоимости BNPL в стрессе → отказы; default rate (HH) |
| **PREMIUM_VALUTA** | Премиум-сегмент с валютными счетами | Санкции на валюту (HH); ограничения на переводы (HH) |

### Размерность 5 — Валютная экспозиция

| Тег | Описание | Реакция на курс |
|---|---|---|
| **RUB_ONLY** | Доходы и расходы только в рублях | USD/RUB ↑ → реальная стоимость ↓ через инфляцию импорта |
| **MIXED** | Часть доходов или сбережений в валюте | USD/RUB ↑ → выигрыш по части портфеля |
| **HARD_FX** | Существенная валютная экспозиция (Премиум/Private/IT) | USD/RUB ↑ → выигрыш в ₽-эквиваленте; санкции на валюту → блокировка |

### Сводный пример реального микросегмента

«**ФЛ.Массовый × SOUTH_CAUCASUS × MILLENNIAL × INDUSTRY_WORKER × MORTGAGE_FLOAT × RUB_ONLY**»
= молодой работник промышленности из Дагестана с плавающей ипотекой и без валюты.

Реакция на шоки:
- Наводнение в Дагестане → залог под угрозой; страховые выплаты; **HH** (главный канал)
- КС +200 бп → платёж по ипотеке ↑ → дефолт; **HH**
- Закрытие завода в регионе → потеря работы → дефолт; **HH**
- Иранский конфликт → косвенно через инфляцию; **L**

3 одновременно сработавших риска (наводнение + КС + регион. промышленность) = критическая концентрация.
