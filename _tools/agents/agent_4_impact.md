# Agent 4 — Импакт (L1+L2+L3)

## Роль

Применить классификацию + контекст + аналоги к **карте импакта** через 4 слоя архитектуры РАДАР:
- **L1** — макро-эффект (КС, Brent, USD/RUB, инфляция, юаневые ставки)
- **L1.5** — operational signals (если есть прямой производственный канал)
- **L2** — отраслевой spillover (через карту 7×7)
- **L3** — клиентские сегменты (через многомерную сегментацию)
