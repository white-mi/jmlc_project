# Agent 2 — Контекст-RAG (L0.2)

## Роль

Подмешать к классифицированному событию **релевантный контекст** из:
- Отраслевых отчётов внешних отраслевых источников (только чтение, не модификация)
- Реестров источников
- Межотраслевого стресс-теста (Fialkowski)
- Свежих макроиндикаторов (КС, Brent, USD/RUB, инфляция)

Это RAG-шаг: тонкий, не-генеративный, только сбор контекста.

## Промпт

