### ⚠ Акрон — row-shift агрегатора (подтверждён повторно)

smart-lab выставлял 257.2/179.5/198.2/237.6 против **2021/2022/2023/2024**. Но YoY-дельты
(−30%/+10%/+20%) совпадают с офиц. релизами Акрона за **2023/2024/2025** → истинный маппинг
**2022=257.2 (рекорд) … 2025=237.6**. Исходный «sanity-range» (234@2022/237@2025) был сам сдвинут —
агенты исправили по первоисточникам (Interfax/acron.ru/rapu.ru). **Акрон-2021 МСФО = GAP** (пауза
раскрытия; «180.57 за 2022» в медиа — это РСБУ, НЕ МСФО, не использовать).

### Источники выручки
- **ФосАгро**: phosagro.com FY2024 PR (2021/2023 comparative), 12M2022 PR; 2025 — вторичка (Tadviser/AKM/smart-lab, первичный PR не индексирован) → med.
- **Акрон**: interfax.ru/business/953100, acron.ru FY2024 PR, rapu.ru FY2025. 2022 реконструировано из дельт → med.
- **КуйбышевАзот**: smart-lab KAZT МСФО + Ведомости (2023). РСБУ-2024=74.6 ≠ МСФО-81.0 — не путать.
