## 2. Модели — структурная ПОДКЛЮЧЕНА (Фаза C)

1. **StructuralOSL — реально работает для химии** (в отличие от нефтегаза). Reuse `osl_chemistry`:
   цена_период = `base × (драйвер / реф_2025)`, где `base` — 2025-калиброванная avg-цена-за-тонну
   из `osl_chemistry` (ФосАгро=543 через `dap`; Акрон/КуйбышевАзот=297 через `urea`; КОС=1500 через
   прокси `crude_brent`), драйвер даёт **per-period вариацию** — как steel-proxy в металлургии.
   Формула выручки `vol×$/т×FX×(1+other)`; скаляр `k` (median actual/raw по train) поглощает
   бленд/домест-экспорт. **Refs и base — фиксированные константы (не из train/test) → без leakage.**
   КуйбышевАзот без объёма → `_raw=None` (graceful, идёт по learned/persistence).
2. **LinearPanel** (ElasticNet/Ridge) + issuer-FE; **GBMPanel** (NaN-нативный); **persistence/issuer_mean**.

База сравнения skill/DM = **`structural_osl`** (предсказывает) — закреплено `test_chemistry_base_is_structural`.

## 3. Walk-forward (out-of-sample)

Expanding-window, фолды 2022→2025. Общий набор (где structural дал прогноз) = 10 строк.

| model | MAPE_common % | skill vs structural | DM p |
|---|---|---|---|
| **structural_osl** | **13.78** | baseline | — |
| elasticnet | 13.12 | +0.048 | 0.92 |
| issuer_mean | 13.50 | +0.021 | 0.97 |
| ridge | 14.61 | −0.060 | 0.90 |
| hist_gbm | 16.11 | −0.169 | 0.76 |
| persistence | 16.10 | −0.168 | 0.77 |

**Главный вывод — третья отличная картина по отраслям.** Структурная доменная модель (13.78%)
**competitive**: бьёт наивный persistence (16.10%), GBM и ridge, статистически неотличима от
лучших learned (elasticnet 13.12, issuer_mean 13.50) — все DM p>0.76 на N=10–16.

| Отрасль | Что побеждает |
|---|---|
| Металлургия (N=24) | наивный persistence (на малом N ничто не отличимо) |
| Нефтегаз (N=18) | learned (волатильность ломает наивное «прошлый=текущий») |
| **Химия (N=18)** | **структурная доменная модель держится** (цены публичны, физика чистая) |

> **Оговорка:** на N=10–24 это направленный паттерн, **не закон** — DM всюду p>0.76 (различия не
> значимы). Показываем, что́ выигрывает по регимам данных, а не «доказанное превосходство».

Это и есть ценность подключённой структурной: на химии физический Q×P-прогноз — не худший, а
один из лучших, и при этом **интерпретируемый** (видно, какая цена/объём двигают выручку).

## 4. Split-conformal (out-of-sample)

proper-train ≤2022 · calib 2023 · test 2024–25. α=0.10.

| Модель | q | OOS-покрытие | примечание |
|---|---|---|---|
| structural_osl | ±10% | 1/6 = 17% | **calib n=2** (разреженный объём: КуйбышевАзот/КОС-2023 без vol) → ненадёжно |
| **elasticnet** | ±20% | 6/8 = 75% | calib n=4 — рабочий интервал |
| persistence / issuer_mean | ±43% | 8/8 = 100% | широко (ловит всё, но неинформативно) |

**Честно:** для интервалов берём **elasticnet** (full-coverage остатки, calib n=4 → ±20%, 75%).
Структурная даёт calib n=2 (объёма нет у половины эмитентов в ранних годах) → её conformal
ненадёжен — это **документированное ограничение разрежённости объёмов**, не дефект метода.
Структурная сильна в точечном прогнозе (§3), интервалы — на learned-модели.

## 5. Анти-leakage

- **Цены контемпоральны** (окно ⊆ периода, `test_price_window_within_reporting_period` покрывает химию).
- **Структурные refs/base — фиксированные константы**, не из train/test (проверено ревью Фазы C).
- **Статистики только train** (scaler/FE/k); сплит по времени; MAPE валюто-инвариантна.

## 6. Честные ограничения

1. **N=18**; КуйбышевАзот без объёма (структурная его пропускает); КОС — объём только 2024–25 →
   его `k` на n=2 (хрупкий, прокси `crude_brent` для полимеров).
2. **На малом N модели статистически неотличимы** (DM p>0.76) — структурная *competitive*, но не
   «доказано лучшая». Показываем как есть.
3. **2025-цены — low confidence** (предварительные World Bank); Акрон-2022 реконструирован из дельт.
4. **Структурный conformal ненадёжен** (разреженный объём) → интервалы на elasticnet.
