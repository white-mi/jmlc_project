## §1. Проблема

**Стандартный кредитный анализ оценивает каждого заёмщика изолированно и потому
систематически занижает системный риск, когда один макродрайвер бьёт сразу по группе
связанных заёмщиков — а узнаёт об ударе из МСФО-отчётности с лагом 2–3 месяца, постфактум.**

| Грань | Оценка | Источник |
|---|---|---|
| Занижение системного риска без учёта цепочек поставок | **+28 … +70 %** | Fialkowski 2025, [arXiv:2502.17044](https://arxiv.org/abs/2502.17044) |
| Занижение на РФ-кейсе «кредитного цунами Мечела» | **30–45 %** | Ведомости 26.12.2025, Forbes, АКРА |
| Лаг МСФО, с которым аналитик узнаёт о падении выручки | **2–3 месяца** | OSL-бэктест |

Контекст-усилитель 2026: КС РФ в зоне острого стресса (>18 %) амплифицирует коррелированные
дефолты — режим, где изолированный стресс-тест ошибается сильнее всего. Полное изложение —
`product/PERSONAS_JTBD.md` §1.

## §3. Конкуренты (сводка)

Полный разбор — `product/COMPETITORS.md`. Главный честный тезис: **дифференциатор НЕ
«у нас есть цепочки поставок»** (у Bloomberg `SPLC` они уже есть), а **РФ-специфичный
сквозной контур** новость→макро→отраслевой каскад→бифуркация по сегментам банка, без лицензий.

| Класс конкурента | Примеры | Где радар отличается |
|---|---|---|
| Глобальные кредит-платформы | Bloomberg (DRSK/SPLC), Moody's CreditEdge, S&P RiskGauge | РФ-portfolio fit + сегментная бифуркация + цена 0 |
| Энтерпрайз стресс-тест/ECL | SAS Risk Stratum, внутрибанк. CCAR/EBA | каскад по группе vs изолированный по-заёмщику |
| РФ-инкумбенты | СПАРК-Интерфакс, АКРА/Эксперт РА/НКР, ЦБ РФ | макро→отраслевой каскад vs скоринг одной компании; цель — *опережать* рейтинг |

**Позиция:** «контур поверх стандартного стресс-теста — **не заменяет, расширяет**».
Где радар **проигрывает** (честно): покрытие (~28 эмитентов vs млн), зрелость (эксперимент,
нет SLA), реальные данные клиента (L3 `confidence='low'`), spillover пока эвристика ×1.30.

## §5. MVP — что уже работает и где границы

**MVP-статус.** v0.7 (апрель 2026) закрыл MVP — все 4 слоя дают числа за один прогон;
текущая v0.9.x — устойчивость + методология (`CHANGELOG.md`, `_tools/ROADMAP_v0.8.md`).

**Что работает (сквозной контур):**
- L0→L3 за один прогон: новость → макро-состояние РФ → отраслевой каскад → сегменты банка.
- 7 отраслей (**4 с валидированным OSL + conformal, 3 иллюстративных** — тиринг по доступности
  данных, см. [COVERAGE_TIERS](COVERAGE_TIERS.md)), ~28 эмитентов, интервальный прогноз выручки.
- L0 — 5-агентный LLM-pipeline + RAG; маршрутизация 27 подкатегорий шоков.
- Бифуркация и стратификация на уровне клиентских подгрупп; прослеживаемость до канала/числа.
- Инженерия: **254 pytest зелёных**, CI (GitHub Actions), ruff/black, 4 живых макрофида.

**Границы MVP (явно, из `docs/3_User_Guide_Analyst.md` §7):**
- ❌ не real-time (ручной запуск); ❌ L3 не калиброван на данных банка (`confidence='low'`);
- ❌ не прогнозирует курсы/цены (реагирует); ❌ не работает на отдельной SME-компании;
- ❌ не выдаёт вердикт без интерпретации аналитиком.

Это **зрелость, а не вапорвейр**: ядро прогноза валидировано (`DS_REPORT`), надстройки
реализованы и тестируются, но часть — некалибрована (см. §4, §6).

## §6. Импакт — бизнес-ценность на примере «Мечел-2025»

> ⚠️ **Дисклеймер.** Ниже — **ретроспективная иллюстрация** механизма: «что радар *показал
> бы* и когда», а не пойманный вживую кейс (радар не работал в проде в 12.2025).
> Денежные величины — либо **публичные** (132 млрд / 10,4 млрд по Мечелу), либо
> **условные-иллюстративные** (помечены явно); данных портфеля Т-Банка — **ноль**.

### Таймлайн: сигнал → каскад → решение банка

**1. Опережающий сигнал (за 2–3 мес. до отсрочки 12.2025).**
Операционные данные металлургии (производство × цена FOB/LME × FX) обновляются ежемесячно и
до МСФО. OSL по стали даёт прогноз падения выручки **раньше** публикации отчётности
(механизм H1: точечная точность валидирована **OOS** — `DS_REPORT`; сам лид-тайм 2–3 мес. —
из OSL-бэктеста. Точность на реальных МСФО: НЛМК 0.7 %, ММК 3.1 % MAE). Для Мечела факт —
выручка −26 %, убыток 10,4 млрд ₽, отсрочка **132 млрд ₽**, АКРА → «негатив» (постфактум).

**2. Каскад по группе (общий драйвер, не три отдельных заёмщика).**
Драйвер — КС 21 % + сжатие спроса — синхронно ударил металлургов, застройщиков и непрод-розницу
(«кредитное цунами», Forbes). Изолированный стресс-тест занизил бы риск на 30–45 %. Радар
применяет к **группе скоррелированных заёмщиков** усиление: ×1.30 (эвристика Fialkowski) или
**×1.50** для наблюдаемого кейса Мечела (по публичным источникам). Реальная экспозиция группы при общем
шоке — в **1,28–1,70 раза выше** суммы изолированных лимитов.

**3. Включаемое решение банка.**
Ранний пересмотр лимитов/резервов по **группе** (металлурги + связанные застройщики/ритейл),
а не по одному имени; постановка триггеров мониторинга; действие **до** даунгрейда АКРА.

### Иллюстративная количественная ценность

| Рычаг | Иллюстрация (помечено условным) | Откуда |
|---|---|---|
| Лид-тайм | сдвиг решения с «после МСФО» на **−2…−3 мес.** (мес → дни до рейтинг-действия) | H1, OSL-лид-тайм |
| Полнота экспозиции | учёт группового усиления **×1.30–1.50** (*эвристика, не калибр. на РФ*) вместо заниженной суммы лимитов | Fialkowski; кейс Мечел ×1.50 |
| Экономия на одном кейсе | *иллюстративно* **50–200 млн ₽** за избежание одной просадки рейтинга крупного заёмщика | OSL-бэктест (помечено иллюстративным) |
| Время аналитика | вердикт «шум vs критично» за минуты вместо ручного разбора | целевая метрика пилота (§5 `BACKTEST_FEEDBACK`) |

**Честная рамка ценности:** лид-тайм и сегментная бифуркация — это **измеримая** часть (H1–H3,
OOS). Денежная оценка «50–200 млн ₽» и групповое ×1.50 — **иллюстративные/некалиброванные на
РФ** (H4 partial). Перевод ×1.30 из «европейской эвристики» в «локально валидированный» —
план Фазы 4 (COVID-2020 / санкции-2022 / ставка-12.2024). Реальная пользовательская ценность
будет измерена пилотом (метрики — `product/BACKTEST_FEEDBACK.md`), а не заявлена.

## §7. Обратная связь (сводка)

Полно — `product/BACKTEST_FEEDBACK.md`. Честно: **пользовательских интервью/пилота нет**
(forward-план с гайдом и метриками — готов). Зато есть **обратная связь от данных**:
- Воспроизводимый трек-рекорд: 18 разборов (2026-03-27…2026-05-07), `python backtest_analyses.py`.
- Реальный цикл «промах → фикс»: нефтегаз-провал (НДПИ) → формула НДПИ+демпфер; Норникель
  −13.7 % → byproducts; in-sample conformal → split-conformal на независимой панели.
- Три слоя честности прогноза: v0.1 «первое касание» → v0.6 калибровка (**in-sample**) →
  walk-forward (**единственный out-of-sample**, `DS_REPORT`).
