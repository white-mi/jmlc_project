# Макро-радар

### Новость → макро РФ → отрасль → **кредитный риск сегмента банка**
### на **2–3 месяца раньше**, чем эффект дойдёт до отчётности

ITMO Junior ML Contest · публичный репозиторий · **233 теста, 0 skipped**

---

## Проблема

Классический стресс-тест скорит каждого заёмщика **в изоляции**.

Он пропускает доминирующий механизм кризиса: один макро-драйвер (шок ставки,
обвал спроса) бьёт по **коррелированной группе** заёмщиков сразу — через общие
цепочки поставок.

> Fialkowski et al. (2025, arXiv:2502.17044): эта слепая зона —
> **+28…+70 % системного риска**.

---

## Идея: один прослеживаемый путь

```
новость → макро-состояние РФ → межотраслевой каскад → сегмент клиентов банка
```

Каждое число привязано к **каналу передачи** и **публичному источнику** —
аналитик может *защитить* вывод, а не доверять чёрному ящику.

**Выход:** ΔPD / Δспрос / Δотток по сегментам.

---

## Архитектура — 5 слоёв

| Слой | Что делает |
|---|---|
| **L0** Фильтр новостей | 5 LLM-агентов + RAG → 27 подкатегорий шоков |
| **L1** Макро-состояние | РФ-CAI · EPU · режим ключевой ставки |
| **L1.5** Операционный сигнал | выручка = объёмы × цены × FX, **раньше МСФО** + conformal |
| **L2** Spillover | матрица отраслей 7×7 (Fialkowski) · кредитный канал |
| **L3** Сегменты | ΔPD/Δспрос/Δотток · 18 сегментов · 5 каналов |

---

## Что отличает: Data Science уровня, а не демо

**4 отрасли** проработаны глубоко на **реальных панелях FY2021–2025**
(только публичные МСФО + биржевые цены):

- честная **out-of-sample** walk-forward (expanding-window, сплит по времени)
- **split-conformal** интервалы (finite-sample)
- наивные бейзлайны (persistence / issuer-mean) + Diebold–Mariano
- анти-leakage аудит на каждом шаге

---

## DS-результат — честный, без приукрашивания

| Отрасль | N | Кто выигрывает по MAPE |
|---|---|---|
| Металлургия | 24 | наивный persistence |
| Нефтегаз | 18 | learned (волатильность) |
| Химия | 18 | структурная competitive |
| Энергетика | 30 | learned/persistence > структурной |

**Вывод, который мы *показываем*, а не прячем:** на малом N различия
**статистически не значимы** (DM p>0.14). Победитель зависит от **регима данных** —
это паттерн, не закон.

---

## Где операционный сигнал реально выигрывает

Энергетика, split-conformal (прогноз 2024–25 по данным ≤2022):

| Модель | OOS-покрытие |
|---|---|
| **структурная (физсигнал года t)** | **12/12 = 100 %** |
| persistence / learned (stale ≤2022) | 2/12 = **17 %** |

Авторегрессия «застревает» на старом уровне. Структурная **читает
контемпоральные физданные** (генерацию, цены РСВ/КОМ — на 2–3 мес. раньше МСФО)
→ отслеживает тренд. **Это и есть ценность L1.5.**

## Интеллектуальная честность = зрелость

Не «красивые цифры», а воспроизводимая правда:

- `/doublecheck` + `/fact-check` поймали **6 битых ячеек** агрегаторов (сдвиги рядов)
  и структурный разрыв (деконсолидация ЛУКОЙЛа)
- ревью убрало **раздутые интерсепты** в энергетике (8 % → честные 12 %)
- каждое ограничение **задокументировано**, тесты-стражи против «тихой подгонки»

## Применение ИИ — в продукте и в процессе

**В продукте:** L0 — это 5-агентный LLM-пайплайн + RAG (работает офлайн dry-run).

**В процессе разработки:**
- построено в Claude Code
- мульти-агентный сбор данных + `/doublecheck`/`/fact-check` верификация
- субагенты-ревьюеры с гейтом **≥9/10** после каждой фазы каждой отрасли

Аутентичный агентный воркфлоу с прослеживаемым аудит-трейлом.

## Честные ограничения (фичи, не баги)

- DS глубоко на 4 из 7 отраслей; малый N → выводы с DM/skill, не «доказанная точность»
- L3 не калиброван на данных банка (`confidence=low`, экспертные приоры)
- ×1.30 spillover-усилитель — эвристика Fialkowski, не калибрована на РФ-шоках
- теплосегмент энергетики вне структурной модели
