# EDA — импликации для модели (chemistry)

- **01** Динамика выручки: индексация снимает разницу масштаба/валюты. Видно расхождение траекторий эмитентов внутри отрасли. Импликация: нужны issuer fixed-effects (своя база у каждого эмитента).

- **02** Эластичность выручка↔цена. ФосАгро/DAP (фосфор): эласт.≈0.79 (n=5); Акрон/карбамид: эласт.≈0.45 (n=4); КуйбышевАзот/карбамид: эласт.≈0.29 (n=5); КОС/нефть Brent (прокси полимеров): эласт.≈0.40 (n=4). Структурная модель неявно предполагает наклон=1 (выручка=Q×P); отклонения эмпирического наклона (хедж, лаги, доля рынка, бленд продуктов) — то, что learned-модель может поймать.

- **03** FX pass-through: ПРОПУЩЕНО — у отрасли нет USD-ценовой сырьёвой корзины для реконструкции «реализованного курса». Фигура осмысленна только там, где выручка считается через USD-цены товара (металлургия). Импликация: здесь FX входит как обычный ценовой признак (usd_rub), отдельная реконструкция курса не нужна.

- **04** Объём↔выручка (нормированы). Где связь слабее «чистого» Q×P — значит цена и микс важнее объёма в эти годы (объём падал, выручка держалась на цене). Импликация: цена обязана быть признаком, объёма недостаточно.

- **05** Пропуски: 17% ячеек драйверов пусты (объёмы — 69%). Структурно объёмы разрежены (своя номенклатура у эмитента), а общий для всех заполненный признак — ЦЕНЫ. Импликация: разреженные объёмы → GBM с нативным NaN-handling, а не impute; цены — стабильный костяк.

- **06** Масштаб таргета: от 80.9 до 574 — разброс уровней эмитентов. log-преобразование выравнивает дисперсию. Импликация: моделируем log(выручка); метрика MAPE (валюто-инвариантна); issuer-FE снимает межуровневый сдвиг.

- **07** Мультиколлинеарность цен (n=5 периодов). VIF (сокр. набор dap/urea/phosphate_rock/crude_brent, n>p): dap=7.1, urea=7.5, phosphate_rock=3.0, crude_brent=5.1 (>5-10 ⇒ сильная коллинеарность). Макс|corr|=0.92, cond(матрицы цен)≈2e+15 ⇒ практически вырождена (раздута малой выборкой: 5 цен ≥ 5 периодов). Импликация: высокие VIF/|corr| ⇒ OLS неустойчив → регуляризация (ElasticNet/Ridge) оправдана; на малом N плюс отбор/сжатие цен.

- **08** Бейзлайн StructuralOSL (IN-SAMPLE, per-period цены/объёмы + скаляр k): средн |ошибка| ≈4.6% по 3 эмитентам (11 строк). Это планка структурной физики Q×P, которую честная out-of-sample learned-модель должна побить (Stage D). NB: k фитится in-sample → оценка оптимистична; реальный OOS-разрыв виден в walk-forward. Тонкие эмитенты (n≤2: КОС) дают почти точный in-sample фит — их низкая ошибка не информативна.
